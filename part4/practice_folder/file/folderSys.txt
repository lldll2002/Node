app.js와 같은 레벨에 file 이라는 폴더가 있다.  --> 폴더 경로를 찾는다. -> 경로를 어떻게 찾지?
이 폴더 안에 video, image, txt 폴더를 만들어서  --> 폴더를 새로 만든다. -> 각 이름으로 된 새 폴더는 어떻게 만들지?
.mp4 파일은 video 폴더에, .png 파일은 image 폴더,
.txt 파일은 text 폴더에 넣어주는 시스템을 만들어주세요. --> 이동도 시켜줘야겠다. -> 뭘 이동시키지? -> 파일을 전부 이동시킨다 -> 파일을 전부 어떻게 탐색하지? -> 어떤 걸 쓰지?

(어느 파일이 어느 폴더로 이동했는지 console로 알려주기)
